STRUCTURES

STRUCTURE OF PAGES

TAG NAME DESCRIPTION
<html> </html> HTML Anything between opening & closing tag is HTML code.
<head> </head> HEAD Contains information (metadata) about the document: titles, scripts, and style sheets.
<title> </title> TITLE Located between head tags, title tags define a document's title that is shown in browser's tab.
<body> </body> BODY Anything between the body tags are displayed inside the main browser window.
<header> </header> HEADER Located in body tags, includes: introductory details, title, logo, search form, author name and other elements.
<nav> </nav> NAVIGATION Provides navigation links.
<footer> </footer> FOOTER Contains information about the author of the section, copyright data or links to related documents.
<article> </article> ARTICLE Include items for a forum post, magazine/newspaper article, blog entry or interactive widget/gadget.
<section> </section> SECTION Generic standalone section of a doc, no specific semantic element to represent it.

For more elements visit: mdn web docs

TEXT

MARKUP TEXT

TAG NAME DESCRIPTION
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> HEADINGS <h1> is used for main headings. <h2> is used for subheadings... etc for more subheadings.
<p> </p> PARAGRAPH Represents a paragraph, can contain images or related content to text.
<b> </b> BOLD Words enclosed in these tags make characters appear bold.
<i> </i> ITALIC Words enclosed in these tags make characters appear italic.
<sup> </sup> SUPERSCRIPT Used for superscript characters such as dates or mathematical concepts 22
<sub> </sub> SUBSCRIPT Used for subscript characters for foot notes or chemical formulas such as H2O
<br /> BREAK The <br /> tag creates a line break inside the middle of a paragraph.
<hr /> HORIZONTAL RULES The <hr /> tag creates a break between themes or sections
and will create a line break.
<strong> </strong> STRONG Element indicates that its content has strong importance. WARNING!
<em> </em> EMPHASIS Indicates emphasis and shows the characters in italic.
<pre> </pre> PRE-FORMATTED TEXT Defines pre-formatted text with a fixed-width font, and preserves both spaces and breaks. Text will display exactly as written int he HTML source code.
<blockquote> </blockquote> BLOCKQUOTE Used for longer quotes that take up an entire paragraph. Browsers will indent contents of a blockquote.
<q> </q> "QUOTES" Element is used for shorter quotes. Browsers will place quotes around the <q> element.
<abbr> </abbr> ABBREVIATION Generic standalone section of a doc, no specific semantic element to represent it.
<cite> </cite> CITE Represents a paragraph, can contain images or related content to text.
<dfn> </dfn> DEFINITION Include items for a forum post, magazine/newspaper article, blog entry or interactive widget/gadget.
<address> </address> ADDRESS Contains details for the author of the page. It can contain address, phone number and/or email.
<ins> </ins>, <del> </del> INSERT & DELETE <ins> Shows content that has been inserted into a document, while <del> shows text that has been deleted from it.
<s> </s> STRIKETHROUGH <s> shows element that is no longer accurate or relevant, but should not be deleted. strikethrough

For more advanced text formatting visit: mdn web docs
For complete list of HTML Tags: W3Schools

LISTS

LISTS

TAG NAME DESCRIPTION
<ol> </ol> ORDERED LIST <ol> elements make each <li> a numbered list.
<ul> </ul> UNORDERED LIST <ul> elements make each <li> a bulleted list. List attributes are indented by default. It is best to use the CSS list-style-type property to modify the list specific attributes.
<li> </li> LIST <li> are list elements and are nested in a <ol> or an <ul> element.
<dl> </dl> DEFINITION LIST Consists of terms and their definitions.
<dt> </dt> DEFINITION TERM Contains the term to be defined.
<dd> </dd> DEFINITION Details the definition of a specific term.

For more list formatting options visit: TutorialBrain
More list attributes: mdn web docs

LINKS

LINKS

TAG NAME DESCRIPTION
<a> </a> ABSOLUTE URL The <a> element contains the value of the href attribute, which is the page you want users to visit when they click on the link. URL stands for Uniform Resource Locator. When linking to the same site, shorthand can be used to give the relative URL. See links template
mailto: EMAIL mailto: is used in the <a> element and addresses an email to a specified email address.
target="_blank" TARGET To open a link in a new window the target attribute follows the href attribute, used in the <a> element.
href="#" LINK TO SAME PAGE Used in the <a> element to link to a specific spot on the page. It is referenced to an id="" attribute. See links template
HREF="someurl.com/#bottom" LINK TO ANOTHER PAGE Add the id of another page to link specifically to that location. The id will link to the specified location. See links template

For more hyperlink attributes & colors: W3Schools
Another resource for creating hyperlinks: mdn web docs


LINKS TEMPLATES

            Mailto:

            <p><a href="mailto: enterEmailAddress_removeSpace">Contact</a></p>
            

            Anchor attribute with target (new window):

            <a href="link" target="_blank">someURL<a>
            

            Link to specific location on webpage:

            <nav class="className">
              <ul>
                <li><a href="#id1">id1</a></li>
                <li><a href="#id2">id2</a></li>
                <li><a href="#id3">id3</a></li>
              </ul>
            </nav>
            

            Place in desired location of webpage:

            <h2 id="id1">id1Name</h2>
            

            Link to specific part of another page (if it has an id attribute for that location):

            <li><a href="someURL.com/#link">linkName</a></li>

IMAGES

IMAGES

TAG NAME DESCRIPTION
<img src="imageLink" alt="alternateTextDescription"/> IMAGE The <img /> attribute is a self closing tab, that allows for adding images to the webpage. It carries the src & alt attributes. A title="" attribute can be added to give additional information about the image.
width="300" height="150" WIDTH & HEIGHT Width & height attributes can be included in the <img /> attribute or written in css stylesheet.
<figure> </figure> FIGURE Images that contain a caption are nested in the <figure> element.
<figcaption> </figcaption> FIGCAPTION The <figcaption> element allows the option to add a caption for the image and is nested within the <figure> element.

For more image attributes: W3Schools
Another resource for embedding images in HTML: mdn web docs


IMAGES TEMPLATES

          Image element (without title):

          <img src="imageLink" alt="alternateTextDescription"/>
          
          

          Image element (with title):
          
          <img src="imageLink" alt="alternateTextDescription" title="imageTitle"/>

          

          Image element (sizing attributes):
          
          <img src="imageLink" alt="alternateTextDescription" title="imageTitle" width="200" height="150"/>

          

          Figure & figcaption:
          
          <figure>
            <img src="imageLink" alt="alternateTextDescription"/>
            <br />
            <figcaption>imageDescription_Caption</figcaption>
          </figure>

TABLES

TABLES

TAG NAME DESCRIPTION
<table> </table> TABLE This element holds all content for all HTML tables.
<thead> </thead> TABLE HEAD Set of rows defining column headers in table.
<tr> </tr> TABLE ROW Row of table, contains <td> data </td>.
<td> </td> TABLE DATA Data (or cell) inside a row, nested in <tr> </tr> tags.
<tf> </tf> TABLE FOOT Table footer row.
colspan COLUMN SPAN Sets number of columns <td> element should span.
rowspan ROW SPAN Sets number of rows <td> element should span.

FORMS

FORMS

TAG NAME DESCRIPTION
<form> </form> FORM Controls for forms live inside <form> element, and usually contains a method and id attribute.
action="http://www.notarealLink.com/subscribe.php" ACTION Action states the URL location that information received in the form will be when submitted.
method="get" || method="post" METHOD Method will have two methods: get or post. get method is ideal for short forms or retrieving data from web server. post is ideal for long forms, user needs to upload a file, contains confidential data (such as passwords), and add/edits/deletes information from a database.
<input type="text" name="username" size="8" maxlength="20" /> INPUT The <input> element uses various form controls.
<textarea> </textarea> TEXT AREA Text area creates multi-line text input and should have an opening and closing tag.
<select> </select> SELECT Select allows a drop down list box for users to select one option.
<option> </option> OPTION Option is nested between the <select> element tags and include details on choice options.

Form Controls
adding text: text input, password input, text area
options: radio buttons, checkboxes, drop-down boxes
submitting: submit buttons, image buttons, file upload
